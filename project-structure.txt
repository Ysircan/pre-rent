📁 pre-rent/ → 项目根目录，是一个以原生前端实现的静态多页面网站。
  📁 assets/ → 存放站点使用的静态资源文件。
    📁 images/ → 存放页面与公共头部引用的图片素材。
      📄 44bc39668252b828650b6368b4c9a0e.png → 网站品牌 Logo 图片，供首页和公共导航区域展示。
  📁 css/ → 存放全站样式文件，按公共样式与页面样式拆分。
    📄 base.css → 提供全局重置、基础字体和页面容器的通用样式。
    📄 header.css → 定义公共头部、导航菜单和移动端菜单的样式。
    📁 homepage/ → 存放首页专用的大型展示模块样式。
      📄 homepage.css → 定义首页轮播 Hero、服务介绍、流程、FAQ 等模块的完整视觉样式。
    📁 pages/ → 存放各独立页面的专属样式。
      📄 book.css → 为预约页面准备的独立样式稿，内容与 `book.html` 的内联样式相近。
      📄 home.css → 提供租前检查页和部分首页通用的 Hero、卡片区块、Who 区块与 CTA 样式。
      📄 minimum.css → 定义最低标准检查页的沉浸式布局、滚动卡墙和中央信息面板样式。
      📄 professional.css → 预留给专业服务页的样式文件，当前基本为空占位文件。
      📄 room.css → 定义房间适配性页面的拖拽网格、控制面板和移动端适配样式。
  📁 js/ → 存放站点的前端交互脚本。
    📄 header.js → 负责注入公共头部、切换移动端菜单、设置当前导航高亮，并处理 Who 区块的悬浮文字交互。
  📁 partials/ → 存放可复用的 HTML 片段。
    📄 header.html → 定义全站复用的页头导航结构，供脚本动态加载到各页面。
  📁 rent/ → 存放租前检查相关的专题页面。
    📄 index.html → 租前检查服务总览页，介绍检查类别、适用人群和预约入口。
    📄 minimum-standards.html → 最低标准检查专题页，通过标签切换和动画背景展示检查重点。
    📄 professional-service.html → 专业服务介绍页，目前仅保留标题和基础页面骨架。
    📄 room-suitability.html → 房间适配性评估页，提供可拖拽家具和尺寸模拟功能。
  📄 book.html → 预约页面，展示预约入口切换按钮和可获得的检查内容说明。
  📄 index.html → 网站首页，面向墨尔本生活支持服务展示多个业务入口、轮播 Hero 与说明区块。
  📄 test.html → 专业服务页面的独立原型测试文件，用于验证视觉排版与文案布局。
  📄 test1.html → 首页的独立原型测试文件，内联了首页大部分样式与脚本用于快速预览。

项目总结：
技术栈：原生 HTML、CSS、JavaScript，采用静态多页面结构，并通过 `fetch` + HTML partial 的方式复用公共头部。
核心功能：项目主要用于展示墨尔本生活支持与租前检查服务，包含首页业务介绍、租前检查分类说明、最低标准检查交互展示、房间适配性拖拽模拟、预约入口和响应式导航。
